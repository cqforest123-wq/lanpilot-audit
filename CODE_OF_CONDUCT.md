# Code of Conduct

LANPilot Audit contributors are expected to keep project spaces respectful, professional, and focused on authorized network governance.

## Expected Behavior

- Be kind and constructive.
- Assume good intent while staying precise about safety boundaries.
- Respect different experience levels and language backgrounds.
- Keep discussions focused on lawful, authorized use.

## Unacceptable Behavior

- Harassment or personal attacks.
- Requests or contributions that enable unauthorized access, credential testing, or configuration tampering.
- Publishing private audit data, credentials, or personal information.

## Enforcement

Maintainers may edit, close, or remove issues, pull requests, comments, or contributions that violate this code or the project's safety boundary.
