# App Store Listing - English

## Name

LANPilot Audit

## Subtitle

Authorized local network governance

## Description

LANPilot Audit helps small businesses document LAN assets, reachable services,
client isolation posture, and remediation priorities. Every real audit requires
explicit authorization and follows a fixed local workflow.

Audit evidence remains on the Mac. The app does not upload reports, test
credentials, perform unauthorized login, or change network configuration.
