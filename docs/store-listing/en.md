# LANPilot Audit
- Subtitle: Authorized LAN governance, locally
- Short description: Run a guided, local-first LAN governance audit and turn evidence into a risk register and remediation roadmap.
- Long description: LANPilot Audit helps authorized small-business operators document point-in-time LAN exposure using a fixed, low-intensity workflow. Evidence and reports remain on the Mac.
- Keywords: LAN, governance, audit, risk register, remediation, local-first
- Privacy note: No cloud upload. The user controls local export.
- Safety note: Fixed checks only after explicit authorization; no credential testing or configuration changes.
- Review note summary: Authorized governance assistant using allowlisted local steps.
- Screenshot captions: Authorization; Engine readiness; Audit progress; Findings; Export.
