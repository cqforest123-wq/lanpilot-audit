# LANPilot Audit
- サブタイトル: ローカル優先の承認済みLANガバナンス監査
- 短い説明: ガイド付きのローカル監査で、証跡をリスク台帳と改善ロードマップに整理します。
- 詳細説明: 許可を得た小規模事業者向けに、固定された低負荷の手順でLANの時点観測を記録します。データはMac内に保持されます。
- キーワード: LAN, ガバナンス, 監査, リスク台帳, 改善, ローカル
- プライバシー: クラウド送信なし。エクスポートは利用者が管理します。
- 安全性: 明示的な承認後のみ固定チェックを実行し、認証情報のテストや設定変更は行いません。
- 審査メモ: 許可リスト方式のローカル手順を使う承認済みガバナンス支援アプリです。
- スクリーンショット: 承認確認；エンジン状態；監査進行；所見；エクスポート。
