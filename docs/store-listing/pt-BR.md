# LANPilot Audit
- Subtítulo: Governança LAN autorizada e local
- Descrição curta: Auditoria local guiada com registro de riscos e roteiro de correção.
- Descrição longa: Documenta observações pontuais autorizadas por um fluxo fixo e de baixa intensidade. Os dados permanecem no Mac.
- Palavras-chave: LAN, governança, auditoria, riscos, correção, local
- Privacidade: Sem envio à nuvem; exportação controlada pelo usuário.
- Segurança: Verificações fixas após autorização, sem teste de credenciais ou mudanças de configuração.
- Resumo para revisão: Assistente autorizado com etapas locais permitidas.
- Capturas: Autorização; mecanismo; progresso; achados; exportação.
