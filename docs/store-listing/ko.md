# LANPilot Audit
- 부제: 로컬 우선의 승인된 LAN 거버넌스 감사
- 짧은 설명: 안내형 로컬 감사로 증거를 위험 등록부와 개선 로드맵으로 정리합니다.
- 자세한 설명: 승인된 소규모 사업자 네트워크를 위해 고정된 저강도 절차로 시점 관찰을 기록하며 데이터는 Mac에 보관됩니다.
- 키워드: LAN, 거버넌스, 감사, 위험 등록부, 개선, 로컬 우선
- 개인정보: 클라우드 업로드 없음. 내보내기는 사용자가 제어합니다.
- 안전: 명시적 승인 후 고정 검사만 실행하며 자격 증명 테스트나 설정 변경을 하지 않습니다.
- 심사 요약: 허용 목록의 로컬 단계만 사용하는 승인형 거버넌스 도우미입니다.
- 스크린샷: 승인 확인；엔진 준비；감사 진행；발견 사항；내보내기。
