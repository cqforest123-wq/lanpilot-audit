# LANPilot Audit
- Sous-titre : Gouvernance LAN autorisée et locale
- Description courte : Audit local guidé avec registre des risques et feuille de route corrective.
- Description longue : Documente des observations LAN ponctuelles et autorisées via un flux fixe et peu intensif. Les données restent sur le Mac.
- Mots-clés : LAN, gouvernance, audit, risques, remédiation, local
- Confidentialité : Aucun envoi cloud ; export contrôlé par l’utilisateur.
- Sécurité : Contrôles fixes après autorisation, sans test d’identifiants ni modification de configuration.
- Note de révision : Assistant de gouvernance autorisé utilisant des étapes locales approuvées.
- Captures : Autorisation ; moteur ; progression ; constats ; export.
