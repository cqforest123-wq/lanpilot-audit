# LANPilot Audit
- 副标题：本地优先的授权局域网治理审计
- 简短描述：通过引导式本地审计，将网络证据整理为风险台账和整改路线图。
- 详细描述：LANPilot Audit 面向获得授权的小型企业网络管理人员，以固定、低强度流程记录时点网络暴露情况。证据和报告保留在 Mac 本机。
- 关键词：局域网，网络治理，审计，风险台账，整改，本地优先
- 隐私说明：不上传云端，导出由用户控制。
- 安全说明：仅在明确授权后运行固定检查，不进行凭据测试或配置修改。
- 审核说明摘要：仅调用白名单本地步骤的授权治理助手。
- 截图标题：授权确认；引擎就绪；审计进度；风险发现；本地导出。
