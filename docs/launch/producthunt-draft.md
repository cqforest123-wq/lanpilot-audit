# Product Hunt Draft

## Name

LANPilot Audit

## Tagline

Local-first LAN governance audit assistant for macOS.

## Description

LANPilot Audit helps small teams understand local network exposure, produce governance reports, track remediation, and retest authorized changes without uploading audit evidence to the cloud. It is designed around fixed authorized checks, multilingual reports, raw evidence preservation, and a strong safety boundary.

## Safety Note

No credential testing, no exploit modules, no unauthorized login, no configuration changes, and no cloud upload.
