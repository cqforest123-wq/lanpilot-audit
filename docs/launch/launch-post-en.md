# Launch Post - English

I am opening LANPilot Audit: a local-first LAN governance audit assistant for macOS.

It is built for small-business networks where teams need visibility into assets, service exposure, gateway posture, client isolation, Web/TLS basics, and remediation tracking without sending audit evidence to a cloud service.

Safety boundary:

- No credential testing.
- No exploit modules.
- No unauthorized login.
- No configuration changes.
- No cloud upload.

The macOS DMG release path supports Developer ID signing, Apple notarization, and SHA-256 verification.

If this kind of local-first governance tooling is useful to you, a GitHub star would help the project reach more maintainers and small teams.
